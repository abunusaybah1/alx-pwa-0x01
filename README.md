A modern movie discovery application built with Next.js, TypeScript, and Tailwind CSS using the MoviesDatabase API.

## Project Description

CineSeek is a responsive web app that allows users to browse movies, view details, and search by year or genre. It demonstrates clean component architecture, proper TypeScript typing, API integration, and state management. The project helps you practice building real-world UI features, handling data fetching, and creating reusable components.

## Learning Objectives

- Understand API documentation and usage
- Implement TypeScript interfaces
- Build reusable React components
- Manage filtering, pagination & state
- Use Tailwind CSS for responsive UI
- Handle loading & error states
- Secure API keys using .env
- Use Next.js API routes to proxy external API requests

## Technologies Used

- Next.js 14 (Pages Router)

- Tailwind CSS
- Font Awesome
- MoviesDatabase API
- Node.js, npm, Git
